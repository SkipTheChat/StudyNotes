# 构建乘积数组 

### 信息卡片 

- 时间： 2019-2-1

- 题目描述：

  ```
  给定一个数组A[0,1,...,n-1],请构建一个数组B[0,1,...,n-1],其中B中的元素B[i]=A[0]*A[1]*...*A[i-1]*A[i+1]*...*A[n-1]。不能使用除法。
  ```

```java
  public int[] multiply(int[] A) {
        int[] a = new int[A.length];
        int res = 1;
      //从右向左乘
        for (int i = 0; i < A.length; i++) {
            a[i] = res;
            res *= A[i];
        }

        res = 1;
        //从左向右乘
        for (int i = A.length - 1; i >= 0 ; i--) {
            a[i] *= res;
            res *= A[i];
        }
        return a;
    }
```
