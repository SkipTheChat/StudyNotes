# 二叉树的镜像 

### 信息卡片 

- 时间： 2019-1-23
- 题目描述：

```
操作给定的二叉树，将其变换为源二叉树的镜像。
输入描述:

二叉树的镜像定义：源二叉树 
    	    8
    	   /  \
    	  6   10
    	 / \  / \
    	5  7 9 11
    	
    	镜像二叉树
    	    8
    	   /  \
    	  10   6
    	 / \  / \
    	11 9 7  5

```



### 参考答案

> 思路

递归。

从根节点开始，交换左右节点。

然后分别对左节点和右节点进行递归。




> 代码

```java
    public void Mirror(TreeNode root) {
        if(root == null) return;
        swap(root);

    }

    private void swap(TreeNode root) {
        if(root == null) return;
        TreeNode temp = root.left;
        root.left = root.right;
        root.right = temp;
        swap(root.left);
        swap(root.right);
    }
```
