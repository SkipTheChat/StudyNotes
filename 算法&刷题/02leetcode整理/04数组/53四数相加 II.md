# [四数相加 II](https://leetcode-cn.com/problems/4sum-ii/)

### 信息卡片

- 时间： 2019-5-4
- 难度：中等
- 题目描述：

```
给定四个包含整数的数组列表 A , B , C , D ,计算有多少个元组 (i, j, k, l) ，使得 A[i] + B[j] + C[k] + D[l] = 0。

为了使问题简单化，所有的 A, B, C, D 具有相同的长度 N，且 0 ≤ N ≤ 500 。所有整数的范围在 -228 到 228 - 1 之间，最终结果不会超过 231 - 1 。

例如:

输入:
A = [ 1, 2]
B = [-2,-1]
C = [-1, 2]
D = [ 0, 2]

输出:
2

解释:
两个元组如下:
1. (0, 0, 0, 1) -> A[0] + B[0] + C[0] + D[1] = 1 + (-2) + (-1) + 2 = 0
2. (1, 1, 0, 0) -> A[1] + B[1] + C[0] + D[0] = 2 + (-1) + (-1) + 0 = 0
```





### 参考答案

> 思路

dfs。超时



> 代码

```java
class Solution {
   
    int count = 0;
    public int fourSumCount(int[] A, int[] B, int[] C, int[] D) {
        dfs(A,B,C,D,1,0);
        return count;
    }

    public void dfs(int[] A, int[] B, int[] C, int[] D,int index,int sum){
        if(index == 5){
            if(sum == 0){
                count++;
            }
            return;
        }
        int[] F = null;
        switch (index){
            case 1:F = A;break;
            case 2:F = B;break;
            case 3:F = C;break;
            case 4:F = D;break;
        }


        for (int i = 0; i < F.length; i++) {
            sum += F[i];
            dfs(A,B,C,D,index + 1,sum);
            sum -= F[i];
        }
    }

}
```



### 其他优秀解答

> 思路

我们以存AB两数组之和为例。首先求出A和B任意两数之和$sumAB$，以$sumAB$为key，$sumAB$出现的次数为$value$，存入$hashmap$中。
然后计算$C$和$D$中任意两数之和的相反数$sumCD$，在$hashmap$中查找是否存在$key$为$sumCD$。
算法时间复杂度为$O(n2)$。



> 代码

```java
class Solution {
    public int fourSumCount(int[] A, int[] B, int[] C, int[] D) {
        Map<Integer, Integer> map = new HashMap<>();
        int res = 0;
        for(int i = 0;i<A.length;i++){
            for(int j= 0;j<B.length;j++){
                int sumAB = A[i]+B[j];
                map.put(sumAB,map.getOrDefault(sumAB,0)+1);
            }
        }

        for(int i = 0;i<C.length;i++){
            for(int j = 0;j<D.length;j++){
                int sumCD = -(C[i]+D[j]);
                if(map.containsKey(sumCD)) res += map.get(sumCD);
            }
        }
        return res;
    }
}
```
