# 反转链表 

### 信息卡片 

- 时间： 2020-1-23
- 题目描述：

```
输入一个链表，反转链表后，输出新链表的表头
```



### 参考答案

> 思路

p1指针用于指向p2前面的元素

p2用于通过p2.next = p1进行反转

p3临时保存p2后面的元素




> 代码

```java
/*
public class ListNode {
    int val;
    ListNode next = null;

    ListNode(int val) {
        this.val = val;
    }
}*/
public class Solution {
    public ListNode ReverseList(ListNode head) {
         if(head == null || head.next == null) return head;
        ListNode p1 = null,p2 = head,p3 = head.next;
        while(true){
            p2.next = p1;
            p1 = p2;
            p2 = p3;
            if(p2 == null) break;
            p3 = p3.next;
        }
        return p1;

    }
    
   
}
```



> 复杂度分析

时间复杂度：O(n)

空间复杂度：O(1)
